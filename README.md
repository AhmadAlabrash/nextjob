# NextJob

NextJob is a modern **job listing platform**, similar to Indeed, built with **Next.js**. It allows users to browse, search, and apply for jobs, while employers can post job listings and manage applications. The platform emphasizes a fast, responsive, and user-friendly interface, leveraging modern front-end and authentication technologies.  

---

## 🚀 Features

- **Job Seeker Features**:
  - Browse and search job listings
  - Filter jobs by location, category, and experience
  - Save favorite jobs
  - Apply to jobs via integrated forms
  - User authentication with Clerk (`UserButton`)

- **Employer Features**:
  - Post and manage job listings
  - View applications from candidates
  - Dashboard for job management

- **Common Features**:
  - Responsive UI for desktop and mobile
  - Interactive components with **Radix UI**
  - Dark mode toggle
  - Social login options
  - Clean, accessible, and modern UI

---

## 💻 Tech Stack

- **Frontend**: Next.js (App Router)  
- **UI Components**: Radix UI, Tailwind CSS, Lucide Icons  
- **Authentication**: Clerk (`UserButton`, `useUser`)  
- **State Management**: React hooks / optional context  
- **Backend**: Next.js API routes (or optional database integration)  
- **Version Control**: Git
